# Book Editorial System

Этот набор файлов нужен для превращения эпизодов в книгу, а не в "чуть более гладкую версию урока".

Порядок работы:

1. Сначала прочитать [01_editorial_manifesto.md](/Users/iuriinovosolov/Documents/SylaSlovaDramma/context/editorial_system/01_editorial_manifesto.md).
2. Затем выбрать агента и взять промпт из [03_prompt_templates.md](/Users/iuriinovosolov/Documents/SylaSlovaDramma/context/editorial_system/03_prompt_templates.md).
3. После каждой правки прогнать результат по [04_review_checklist.md](/Users/iuriinovosolov/Documents/SylaSlovaDramma/context/editorial_system/04_review_checklist.md).
4. Если глава не проходит чеклист, правка не принимается.

Смысл системы:

- мы строим романную детскую прозу с учебным каркасом;
- мы не сохраняем игровой или сценарный формат как главный;
- мы разрешаем себе крупную перестройку сцен, если это делает книгу сильнее;
- мы запрещаем правки, которые только полируют поверхность и оставляют прежнюю проблему.
