# Expanded Drama Practice Rewrites For Episodes 1-7

Это расширенная версия.

Здесь я не трогаю количество практики.
Я даю много конкретных замен для тех вопросов, которые звучат слишком внешне
к драме, но при этом сохраняю учебную идею:
- факт / мнение / неправда;
- проверяемое / мнение / хитрое утверждение;
- закономерность / совпадение;
- причина / совпадение;
- есть инструмент / нет инструмента;
- объяснение через тот же тип проверки.
