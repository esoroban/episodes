# Agents for the "Power of Words" project

6 out of ~130 agents selected from the [agency-agents](https://github.com/msitarzewski/agency-agents) repository.

## Selected

| Agent | File | Why we need it |
|-------|------|----------------|
| **Narrative Designer** | narrative-designer.md | Branching, dialogues, Voice Pillars, 3 levels of lore, narrative-gameplay integration |
| **Narratologist** | academic-narratologist.md | Structural analysis (Propp, Campbell, McKee), tracking narrative debts, checking want/need/lie arcs |
| **Psychologist** | academic-psychologist.md | Psychological authenticity of characters (Big Five, attachment, defense mechanisms, group dynamics) |
| **Game Designer** | game-designer.md | Gameplay loops, onboarding, balance, player psychology — for the interactive lesson layer |
| **Book Co-Author** | marketing-book-co-author.md | Writing workflow: raw material -> structure -> draft -> editing -> iteration |
| **Reality Checker** | testing-reality-checker.md | Quality checks: default is "NEEDS WORK", looks for holes, violations, cliches |

## Not selected

| Agent | Reason |
|-------|--------|
| Level Designer | Physical 3D spaces, not our format |
| Visual Storyteller | Marketing visuals, not screenwriting |
| Corporate Training Designer | Chinese corporate context, not children's education |
| Studio Producer | Portfolio management, we have a single project |
| Historian | Historical accuracy, our world is fantasy |

## How to use

These agents are not Cowork skills, but **reference prompts**. They can be:
1. Turned into skills via skill-creator
2. Used as templates for our own skills
3. Mined for specific templates (Voice Pillars, Character Arc, Beat Sheet) to include in our skills
