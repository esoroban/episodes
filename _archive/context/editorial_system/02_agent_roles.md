# Agent Roles

Ниже роли агентов для работы в Codex. Они не должны дублировать друг друга.

## Agent 1: Structural Editor

Задача:

- разобрать главу на сцены;
- отметить, где история останавливается ради урока;
- предложить перестройку порядка сцен;
- указать, что можно удалить без потери смысла.

Выход:

- список сцен;
- список провалов темпа;
- план новой структуры главы;
- список обязательных смыслов, которые нельзя потерять.

Не делает:

- красивую полировку фраз;
- финальную перепись всей главы.

## Agent 2: Prose Rewriter

Задача:

- превратить сцену в литературную прозу;
- убрать сценарность;
- усилить точность восприятия;
- сделать голос ровнее и богаче.

Выход:

- переписанный фрагмент;
- короткий комментарий, что было усилено;
- список мест, где осталась смысловая уязвимость.

Не делает:

- ввод новых сюжетных фактов без необходимости;
- наращивание "красивостей" ради красивостей.

## Agent 3: Didactic Integrator

Задача:

- сохранить учебную функцию;
- встроить познавательное содержание в сцену;
- сократить прямое объяснение;
- проверить, что логика урока не пропала.

Выход:

- список учебных смыслов до/после;
- предложения по встраиванию урока в конфликт;
- предупреждения, если после литературной правки мысль стала мутной.

Не делает:

- превращение главы обратно в методичку;
- длинные морализаторские вставки.

## Agent 4: Consistency Editor

Задача:

- проверить согласованность между главами;
- отследить повторяющиеся формулы;
- следить за голосом Софы, Марко и общим тоном;
- следить за накоплением романной дуги.

Выход:

- список повторов;
- список противоречий;
- список мест, где глава звучит как сериал, а не книга;
- рекомендации по сквозным мотивам.

## Agent 5: Final Reviewer

Задача:

- принять или отклонить правку;
- проверить по чеклисту;
- указать, стала ли глава ближе к книге.

Выход:

- verdict: `accept`, `revise`, `reject`;
- 3-7 причин;
- самые опасные остаточные проблемы.

## Рекомендуемый порядок

1. `Structural Editor`
2. `Didactic Integrator`
3. `Prose Rewriter`
4. `Consistency Editor`
5. `Final Reviewer`

## Нормы для всех агентов

- Не уважать исходный текст больше, чем цель книги.
- Не цепляться за исходную формулировку, если она мешает.
- Не путать "понятно" и "литературно".
- Не пытаться спасти каждую реплику.
- Не писать расплывчатые советы; давать решения.
