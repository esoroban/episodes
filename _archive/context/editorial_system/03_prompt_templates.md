# Prompt Templates

Ниже готовые шаблоны для запуска работы в Codex.

## 1. Structural Editor Prompt

```text
Ты работаешь как Structural Editor для детской/подростковой книги.

Контекст:
- Мы не полируем эпизод.
- Мы превращаем его в главу книги.
- Учебный каркас должен сохраниться, но не управлять ритмом главы.

Прочитай файл: <PATH_TO_CHAPTER>

Сделай:
1. Разбей главу на сцены.
2. Для каждой сцены определи: сюжетная функция, эмоциональная функция, учебная функция.
3. Отметь места, где сцена ломается из-за объяснения.
4. Предложи новую структуру главы.
5. Укажи, что удалить, что сжать, что переписать с нуля.

Формат ответа:
- Scene map
- Tempo failures
- Keep at all costs
- Cut or compress
- New structure

Не переписывай главу целиком.
Не давай общих советов.
```

## 2. Prose Rewriter Prompt

```text
Ты работаешь как Prose Rewriter.

Контекст:
- Мы строим книгу, а не интерактивный урок.
- Нужна живая, плотная, ясная детская/подростковая проза.
- Нельзя скатываться в сценарность, методичку или украшательство.

Прочитай:
- <PATH_TO_CHAPTER>
- /Users/iuriinovosolov/Documents/SylaSlovaDramma/context/editorial_system/01_editorial_manifesto.md

Перепиши только этот диапазон: <LINES_OR_SECTION>

Цель:
- убрать сценарные маркеры;
- усилить внутреннюю оптику Марко;
- сократить прямую дидактику;
- сохранить сюжет и смысл урока;
- сделать сцену книжной.

После переписи добавь:
- что стало сильнее;
- что все еще требует ручной доработки.
```

## 3. Didactic Integrator Prompt

```text
Ты работаешь как Didactic Integrator.

Контекст:
- Нам важно сохранить образовательный смысл.
- Но он должен жить внутри действия, а не рядом с ним.

Прочитай:
- <PATH_TO_CHAPTER>
- /Users/iuriinovosolov/Documents/SylaSlovaDramma/context/editorial_system/01_editorial_manifesto.md

Сделай:
1. Выпиши учебные смыслы главы.
2. Покажи, где они поданы слишком прямо.
3. Для каждого такого места предложи способ встроить смысл в действие, наблюдение, конфликт или внутреннюю речь.
4. Отдельно отметь смыслы, которые нельзя потерять при литературной правке.

Формат:
- Learning goals
- Too explicit
- Better dramatic embedding
- Non-negotiables
```

Проверь файлы:

## 5. Final Reviewer Prompt

```text
Ты работаешь как Final Reviewer.

Сравни:
- исходный фрагмент;
- новый фрагмент;
- редакторский манифест.

Задача:
- решить, стал ли текст ближе к книге;
- отбраковать правку, если она стала только глаже, но не литературнее.

Формат:
- Verdict: accept / revise / reject
- Stronger now
- Still weak
- Biggest risk

Будь строгим.
```

## 6. Master Prompt For Full Chapter Pass

```text
Мы превращаем главу из /publish/book в полноценную главу книги.

Перед работой:
1. Прочитай редакторский манифест.
2. Не защищай исходный формат.
3. Сначала найди структурные проблемы, потом стилистические.

Цель:
- живая книга;
- встроенная, но не доминирующая дидактика;
- сильный внутренний фокус Марко;
- Софа как персонаж, а не учебный виджет.

Запрещено:
- формат пьесы;
- длинные блоки классификации;
- объяснение того, что уже понятно из сцены;
- поверхностная "красивость" без драматической функции.

В конце ответа дай:
- что переписать сначала;
- что можно править массово;
- где нужен человек, а не агент.
```
