# Drama Practice Rewrites For Episodes 1-7

Задача этого файла:
- выловить практику, которая слабо связана с драмой;
- не сокращать количество квизов;
- предложить замены, которые ближе к миру Марко, Софии, Софы, Лины, Леона, Веры и Голоса.

Принцип:
- если можно, заменяем только оболочку примера, а учебную функцию сохраняем;
- если можно, переносим пример в школу, дом, Зеркальный Город, слухи, дневник, тени, артефакты;
- если можно, используем уже существующие объекты сцены.

## Самые полезные массовые принципы замены

Если нужно быстро перепрошивать практику без переписывания всей сцены:

1. Вместо `фильм`, `трава`, `дом`, `собака`, `книжка`, `комната`
   брать:
- дневник;
- рамки;
- Софу;
- школьный чат;
- Веру;
- Лину;
- Леона;
- тень;
- дверь;
- Зеркало;
- число на стене;
- плакаты Голоса;
- громкоговорители.

2. Вместо нейтральных оценок
   брать:
- реплики персонажей;
- поведение персонажей;
- повторяющиеся действия;
- уже существующие улики.

3. Вместо абстрактных “жизненных” примеров
   брать:
- школу Марко;
- дом Марко;
- коридор;
- класс;
- учительскую;
- двор;
- парк;
- Зеркальный Город;
- подвал Леона.
