# Step 3 — Scaling

After a successful pilot (lessons 1A, 1B, 2A, 2B) — the remaining lessons are processed horizontally by phase.

## Order
1. All remaining lessons go through THEORY-REWRITER + QA-THEORY
2. Then PRACTICE-REWRITER + QA-PRACTICE
3. Then CHAPTER-COMPOSER + QA-CONTINUITY

## Principle
Each phase — horizontal (all lessons), then move to the next phase.
This allows catching inconsistencies between chapters at an early stage.
