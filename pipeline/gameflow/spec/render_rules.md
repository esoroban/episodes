# Render Rules — Правила рендеринга HTML

## Принцип

Gameflow YAML рендерится в интерактивный HTML с тёмной темой. Каждая сцена — карточка, одна на экране. Навигация — графовая (по `next_default`, `next_success`, `next_fail`), не линейная.

## Общий стиль

- Тёмная тема: фон `#0d0d12`, карточки `#16161e`
- Шрифт: Georgia/serif для текста, system sans-serif для UI
- Max-width: 760px, центрирован
- Навигация: фиксированная панель внизу (Назад / Дальше)

## Заголовок эпизода

Компактный: название, урок, термины. Не доминирует на экране.

```
Эпизод 1 — «Утро без сестры»
1A.1
Новые термины: факт, мнение
```

## Заголовок сцены

Минимальный. Видно только номер: `3 / 9`. Вся мета-информация (локация, персонажи, настроение, флаги, тип ветвления) скрыта под раскрывающимся `<details>` с иконкой ℹ️.

**Что скрыто:**
- Локация и время: `Квартира Марко, комната · утро`
- Персонажи: `Марко, Софа`
- Настроение: `тревога`
- Тип ветвления: `🔄 Мягкий тупик`
- Установленные флаги: `🏁 sofa_activated`

## Софа = Telegram

**Главное правило: все сцены, где Софа в `characters_present`, рендерятся как Telegram-мессенджер.**

### Разделение драмы и чата (ОБЯЗАТЕЛЬНО)

Telegram-чат — **только для прямых вопросов/ответов Софы и квизов**.
Драма (narrative, диалоги персонажей, автор) — **отдельные экраны** с будущей иллюстрацией.

| Правило | Описание |
|---------|----------|
| Софа в `characters_present` | Только в quiz-сценах и sofa_block-intro (объяснение терминов) |
| Drama-сцены | Без Софы в `characters_present` → рендерятся как отдельные карточки |
| `who: автор` в чате | Максимум 1 короткая ремарка. Больше — выносить в drama-сцену |
| `correct_logic` | НЕ показывается в чате. Только скрытый `quiz-explanation` |
| Нет дублей | `feedback_success` и `correct_logic` не должны повторять друг друга |
| Разрыв chain | Убрать Софу из `characters_present` ИЛИ сменить `location` |

### Когда включается Telegram-стиль

Условие: Софа в `characters_present` И (есть `dialogue` ИЛИ `unlock_button` ИЛИ quiz-опции с `correct`).

Исключение: сцены с `interactions` list или `followup_interaction` (сложные комбо-сцены) — рендерятся стандартно.

### Живой чат (Live Chat)

**Все сообщения появляются по одному, как в реальном Telegram.**

Порядок:
1. Системные сообщения (author_text) — появляются с коротким fade-in
2. Сообщения Софы — "печатает..." (три прыгающие точки, 1 сек), затем пузырь
3. Сообщения Марко — текст появляется в поле ввода внизу, игрок нажимает ➤ (Send)
4. Quiz — Софа "печатает", появляется вопрос с inline-кнопками
5. Unlock — кнопка появляется после typing-анимации
6. Feedback — после ответа Софа снова "печатает" и отправляет результат

**Кнопка «Дальше» заблокирована** пока не все сообщения показаны.

### Элементы чата

| Элемент | Как рендерится |
|---------|----------------|
| Шапка | `📱 Софа` + `онлайн` справа. Фон: `#17212b` |
| Фон чата | `#0e1621` (Telegram dark) |
| Софа (входящее) | Пузырь слева, `#182533`, имя голубое `#6ab3f3` |
| Марко (исходящее) | Пузырь справа, `#2b5278`, имя зелёное `#85c67e` |
| Автор / нарратор | Системное сообщение по центру, `#6d7f8f`, курсив |
| Другие персонажи | Пузырь слева, имя `#e0a060` |
| Typing indicator | Три прыгающие точки в пузыре слева |
| Input bar | Поле ввода + кнопка ➤ внизу чата (для реплик Марко) |
| `author_text` | Системные сообщения ДО диалога |
| `author_text_after` | Обычный текст ВНЕ чата (рефлексия, нарратив) |

### Quiz = Telegram-бот

Quiz в Софа-сценах рендерится как inline keyboard бота:

- Вопрос → пузырь от Софы (большой, до 90% ширины)
- Варианты → кнопки `#3390ec` (фирменный голубой Telegram)
- Правильный ответ → кнопка зеленеет `#4caf50`
- Неправильный → красная `#ef5350` + зачёркивание, правильная подсвечивается
- Feedback → новый пузырь от Софы (с зелёной или красной полоской слева)
- `correct_logic` → НЕ рендерится в чате (хранится в скрытом `quiz-explanation`). В чате — только `feedback_success`/`feedback_soft_fail` от Софы. Дублирование запрещено

**Кнопки блокируют «Дальше»** до ответа.

### Unlock-кнопка

Для голосовых сообщений и разблокируемого контента:

```yaml
unlock_button:
  text: "🔓 Разблокировать запись"
  reveals:
    type: voice_message
    who: София
    line: "Братик, начни с дневника."
    duration: "0:03"
```

- Кнопка: обводка `#6ab3f3`, скруглённая (20px)
- После клика: меняется на «Разблокировано», полупрозрачная
- Появляется voice message: иконка 🎤, waveform ASCII, длительность
- Блокирует «Дальше» до разблокировки

## Обычные сцены (без Софы)

| Элемент | Как рендерится |
|---------|----------------|
| `author_text` | Параграфы с отступами |
| `dialogue` | Реплики с вертикальной линией слева. Софа: фиолетовая линия. Автор: курсив без линии |
| `author_text_after` | Параграфы после диалога |
| Quiz | Карточка с обводкой, кнопки-опции. Зелёный/красный feedback |
| Choice | Кнопки с обводкой `accent`, вертикально |

## Навигация

### Графовая, не линейная

Каждая сцена имеет data-атрибуты:
- `data-next` — куда по умолчанию (из `next_default`)
- `data-next-success` — куда при правильном ответе
- `data-next-fail` — куда при ошибке
- `data-merge-to` — куда сливается ветка

### Кнопка «Дальше»

Приоритет:
1. `choiceTarget` (если игрок выбрал вариант в choice)
2. `quizResults` → `data-next-success` или `data-next-fail`
3. `data-next` (next_default)
4. Fallback: `currentIndex + 1`

### Кнопка «Назад»

Стек истории (как в браузере). Каждый переход вперёд пушит текущий индекс. «Назад» делает pop. Возвращает туда, откуда пришёл — не на предыдущий индекс.

### Блокировка «Дальше»

Сцена блокирует кнопку «Дальше» если:
- Есть quiz с неотвеченными опциями
- Есть choice без выбора
- Есть unlock-кнопка без клика

### Клавиатура

- `→` и `Space` = Дальше
- `←` = Назад

## Progress bar

Полоска вверху экрана. Считает уникальные посещённые сцены, не линейный индекс.

## End screen

После последней сцены: «Конец эпизода N», название, счётчик правильных ответов.

## Visual brief

Скрыт по умолчанию. Кнопка `🎨 Visual brief ▸` раскрывает блок с мета-данными для арт-дирекции (фон, камера, персонажи, реквизит, атмосфера).

## Мобильная адаптация

- Карточки: уменьшенные отступы
- Quiz-кнопки: вертикально (column)
- Заголовок: уменьшенный шрифт
