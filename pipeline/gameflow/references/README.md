# UI References — Canonical Demo

Эта папка — **якорь** для UI/UX решений по игре. Всё, что правим в рендерере
(`tools/build_game.py`) или в gameflow YAML, должно соответствовать референсу здесь.

## Файлы

| Файл | Что это |
|------|---------|
| `ui_canonical.html` | Эталон UI — Drama, Telegram-чат (mobile + desktop), Telegram-бот квизы, debug-тогл для автора |

## Как пользоваться

1. Открыть `ui_canonical.html` в браузере — локально или по относительному пути.
2. Кнопки **Drama / Telegram / Quiz** в правом верхнем углу — переключают экраны.
3. Кнопка **Автор: ON/OFF** — тогл debug-режима (вычитка vs продакшен).
4. Ширина окна <900px → мобильный Telegram fullscreen; ≥900px → Telegram Desktop с сайдбаром.

## Что именно эталонно

- **Два канала доставки:** Софа — текст (чат-пузыри), автор — voice-over (карточка с меткой «АВТОР», пульсирующая точка)
- **Квизы** — inline-keyboard Telegram-бота под пузырём Софы (светлые бот-кнопки, retry на неправильный ответ)
- **CTA-кнопка** — центрированная пилюля с hint-подписью, переход из чата в drama-сцену
- **Drama-экран** — тёмный фон, иллюстрация-плейсхолдер, voice-over полоса снизу
- **Debug-слот автора** — тонкая рельса остаётся в production (резерв под озвучку), текст скрыт

## Правило

Не «отъезжать» от этого референса при доработке рендерера. Если нужно изменить
UI-паттерн — сначала обновить референс здесь и правило в `pipeline_rules.md`,
потом переносить в `build_game.py`.
