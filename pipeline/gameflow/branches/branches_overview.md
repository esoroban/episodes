## Ключевое правило: Information Continuity

**Главная линия самодостаточна.** Игрок, пропустивший ВСЕ ветки, понимает
весь сюжет. Ветки добавляют ГЛУБИНУ, не СЮЖЕТ.

Проверки qa-branches:
1. Main line self-containment — нет ссылок на branch-only контент
2. Branch-to-main transition — merge target не ломается
3. Flag continuity — branch flags не гейтят main line
4. Forward reference — 5 сцен вперёд от merge не ссылаются на ветку
5. Character introduction — персонажи не появляются впервые в ветке
6. Branch quiz isolation — бонус-квизы не в total_quizzes
7. Narrative coherence — переход при пропуске ветки не рваный
